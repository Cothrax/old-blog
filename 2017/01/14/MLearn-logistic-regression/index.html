Machine Learning Week 3 笔记

逻辑回归（Logistic Regression）虽然被叫做“回归”，但实际上是一种分类算法。通过构造新的函数，分类问题也可以应用梯度下降求解。Andrew针对求凸函数极值给出了一种高级 (cū bào) 的优化，还介绍了用于解决过拟合（Overfitting）的正则化（Regularization）。

逻辑回归

二元逻辑回归

正如引言所说，二元逻辑回归是一类分类问题，和线性回归不同的是，其因变量y只有两个值0或者1，即根据自变量x将数据分为两类。

构造新的假设函数：
$$H(x)=\dfrac{1}{1+e^{-x}}\\ h_{\theta}(x)=H(\theta^Tx)$$
函数$H(x)$叫sigmoid/logistic函数大概长这样：

$h_{\theta}(x)=P(y=1;x;\theta)\in(0,1)$，即在给定$\theta$下，$x$属于分类1的概率。
容易发现：
$$\begin{cases} h_{\theta}(x)>0.5 & \text{ if } \theta^Tx>0\\ h_{\theta}(x)<0.5 & \text{ if } \theta^Tx<0 \end{cases}$$
此时就存在一个x取值的集合，使得$\theta^Tx=0$，这个集合就被称为决策边界（Decision Boundary），在边界的两侧分别代表x属于不同的分类。

在逻辑回归中，最小二乘法类型的代价函数$J(\theta)=\dfrac{1}{2m}\sum_{i=1}^{m}(h_{\theta}(x_i)-y_i)^2$不再适用，因为$h{\theta}(x)$的含义已经变化，实际上，这样得到的代价函数不是凸函数，这意味着不能通过梯度下降来找到最小值。
构造新的代价函数：
$$\begin{align*} g_{\theta}(i)&=\begin{cases}-ln(h_{\theta}(x^{(i)})) & \text{ if } y^{(i)}=1\\ -ln(1-h_{\theta}(x^{(i)})) & \text{ if } y^{(i)}=0 \end{cases} \\ &=-y^{(i)}* ln(h_{\theta}(x^{(i)}))-(1-y^{(i)})*ln(1-h_{\theta}(x^{(i)}))\\ J(\theta)&=\sum_{i=1}^mg_{\theta}(i)& \end{align*}$$
函数$g\{\theta}(i)$大概长这样：

当如果$y=1,h_{\theta}(x)\to 0$，即x有很大可能属于分类0，此时将得到$J(\theta)\to+\infty$的惩罚项，反之亦然。
可以证明 (cǐ chǔ shěng lüè) ，这个代价函数是一个凸函数。

$J(\theta)$的偏导数：
为了实现梯度下降，求出代价函数的偏导数：
$$\dfrac{\partial J(\theta)}{\partial\theta_j}=\dfrac{1}{m}\sum_{i=1}^m (h_{\theta}(x^{(i)})-y^{(i)})x^{(i)}_j$$
会发现实际上，逻辑回归的偏导数与线性回归是相同的，但值得注意的是，$h_{\theta}(x)$的含义不再相同。

正则化

欠拟合/过拟合

当自变量x过少时，会出现欠拟合（Underfitting）的问题，而x过多而m太小，则会导致过拟合（Overfitting）。
比如说，如果一组样本数据$(x^{(i)},y^{(i)})$能被一个二次函数很好地拟合，那么当用一次函数即一条直线取拟合数据时，则不能很好的预测结果；而如果用的次数过高，虽然能准确地匹配样本数据（$J(\theta)\to 0$），但是对于新数据往往不能得到很好的预测值，因为它多出了不必要的弯曲和角度。

正则化

解决过拟合的问题，有两种思路：

  1. 减少不必要的自变量，在上述例子中就是去掉高次项。
  2. 让自变量的值尽量小，使函数更加“平滑”，即进行正则化，在上述例子中，就是让高次项的系数尽量小。

在原有的代价函数中，添加正则化系数：
$$J’(\theta)=J(\theta)+\lambda\sum_{i=1}^n\theta_i^2$$
也就是说，对于$\theta$添加惩罚项，当$\theta$过大时，代价会偏高；这样，在拟合样本数据的同时兼顾到$\theta$的大小；其中$\lambda$叫做正则系数。值得注意的是，当正则系数过大时，正则项“喧宾夺主”，也就是说过分注重与$\theta$的大小而导致欠拟合。还有，就是一般不对$\theta_0$进行正则化。

正则化回归

对添加了正则项的$J’(\theta)$求偏导数，可以得到（$j\not=0）$：
$$\begin{align*} \dfrac{\partial J'(\theta)}{\partial\theta_j}&=\dfrac{1}{m}\sum_{i=1}^m (h_{\theta}(x^{(i)})-y^{(i)})x^{(i)}_j+\dfrac{\lambda \theta_j}{m}\\ \theta_j &= \theta_j-\alpha\cdot\dfrac{\partial J'(\theta)}{\partial\theta_j}\\ \theta_j &= \theta_j(1-\dfrac{\lambda\alpha}{m})-\dfrac{1}{m}\sum_{i=1}^m (h_{\theta}(x^{(i)}) \end{align*}$$
$\theta_0$不做正则化，偏导数与原来相同。

contour绘制等值线

contour(x,y,Z,v)用于绘制二元函数$f(x,y)$的轮廓图，也就是等值线。
这个函数就是先画出有限个点(x,y)，将将函数值相等的点连起来构成等值线
参数x和参数y就是这些点的坐标值，描述了legnth(x)*length(y)个点
Z为这些点的$f(x,y)$值，row(Z)==length(x)而且column(Z)==length(y)
v为一个函数值向量，就是要画出的几条等直线的函数值。
比如说，绘制$f(x,y)=x^2+y^2$的等值线

细节：直接将等值向量v设为0是不能正常绘图的（理论上，应该画出函数值为0的等值线），这可能由于0被看做为某个属性值，将0换成[0, 0]即可。
